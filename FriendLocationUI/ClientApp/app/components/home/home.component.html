Localizador de Amigos

Seja bem vindo, esta e uma aplicacao de teste feito com:

  • ASP.NET Core e C# para cross-platform server-side
  • Angular and TypeScript para client-side
  • Webpack para construcao dos recursos do client-side
  • Bootstrap para layout e estilos

Esta aplicacao serve para cadastrar pessoas com suas localizacoes e obter as tres pessoas mais proximas.
